# Unity 6 Racing Game

A complete 3D open-world racing game built with Unity 6 LTS and Universal Render Pipeline.

## Features

- Large seamless open world with multiple biomes
- Advanced vehicle physics with realistic wheel colliders and suspension
- Multiple vehicle classes with unique handling characteristics
- Comprehensive race modes (Circuit, Sprint, Drag, Drift, Time Trial, etc.)
- Dynamic weather system with day/night cycle
- AI traffic system with intelligent behavior
- Police pursuit system with wanted levels
- Garage system with vehicle customization
- Career progression and economy system
- AAA-quality graphics with URP effects

## Requirements

- Unity 6 LTS
- Universal Render Pipeline (URP)
- New Input System
- Addressables
- TextMeshPro

## Setup Instructions

1. Create a new Unity 6 LTS project with URP template
2. Import the project files
3. Configure Project Settings (Input, Physics, etc.)
4. Load the main scene
5. Configure Addressables groups
